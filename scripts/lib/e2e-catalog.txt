=== WORKFLOW MODES (Delegation: e2e_workflow; .cursor/agents/tester.md §4) ===

  v1_full
    prep-v1 → Load Context → Extract → call_records lint (0) → first UCN
    (all applicable UCN sub-steps; include Step 11 ledger when write applies) →
    read_doc → post-write diff (tester.md §6).

  v1_partial
    A subset: e.g. prep-v1 --skip-rebaseline, or extract+lint only, or UCN-only
    on existing data. DAL-only belongs here. List explicit skips + reasons in report.

  v2_full
    Needs round-1 done: prep-v2 → second Extract → lint → second UCN →
    read_doc → diff (new transcripts + regressions).

  v2_partial
    A narrow slice; document out-of-scope sections in the report.

  full
    All eight harness steps, including Run Account Summary (harness 8);
    diff after each UCN (combine or separate tables in the report).
