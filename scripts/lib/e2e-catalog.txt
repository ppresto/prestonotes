=== WORKFLOW MODES (Delegation: e2e_workflow; .cursor/agents/tester.md §4) ===

  e2e_default
    Reserved-trigger default: same steps as this catalog's HARNESS STEPS (prep → chat through second UCN).
    read_doc + post-write diff (tester.md §6) after each UCN where applicable.

  v1_full
    prep-v1 → Load Context → first UCN (all applicable UCN sub-steps; include Step 11 ledger when write applies) →
    read_doc → post-write diff (tester.md §6).

  v1_partial
    A subset: e.g. prep-v1 --skip-rebaseline, or UCN-only on existing data. DAL-only belongs here.
    List explicit skips + reasons in report.

  v2_full
    Needs round-1 done: prep-v2 → second UCN → read_doc → diff (new transcripts + regressions).

  v2_partial
    A narrow slice; document out-of-scope sections in the report.

  full
    End-to-end through both UCN rounds (harness 1–5) with diffs; same E2E scope as e2e_default,
    explicit name when you need a report label. Does not add Run Account Summary to the harness.
