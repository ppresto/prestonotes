# Task Index

This index keeps ephemeral tasks navigable over time.

## Active Tasks
- Add active task files in `docs/tasks/active/`.

## Archive
- Move completed task files to `docs/tasks/archive/YYYY-MM/`.

## Conventions
- File naming: `YYYY-MM-DD-short-task-slug.md`
- Status markers inside each task file:
  - `[ ] TODO`
  - `[x] COMPLETE`
